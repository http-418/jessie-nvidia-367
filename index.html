Jessie packages for nVidia 367.x drivers

These were rebuilt from the packages intended for 'stretch'/'sid' in Debian SVN.

Instructions

  1. Install the signing key:

    sudo apt-key adv --keyserver keys.gnupg.net \
    --recv-key 9BE98D15

  2. Add a sources.list entry:

    deb "http://www.jones.ec/jessie-nvidia-367/" jessie non-free

  3. Stop your desktop manager and/or exit X11

  4. Install the new packages:

    sudo apt-get install build-essential linux-headers-amd64 \
    nvidia-kernel-dkms nvidia-alternative nvidia-modprobe \
    xserver-xorg-video-nvidia
